# Web App for the Philadelphia Pointing Span Test (PPST)
CS 04400 - Section 2 - Spring 2026

## PPST Application
Our Philadelphia Pointing Span Test (PPST) web application will enable
the digital administration of PPST assessments. The system will allow
medical professionals to administer tests and securely capture valuable
patient data to support future evaluation and treatment. The application
will provide a user-friendly interface designed to improve accessibility,
including for patients who may otherwise have difficulty completing the
test. The PPST app will be built with a strong emphasis on security,
accessibility, and operational efficiency.

## Installation
1. Install Python, Django, Visual Studio Code and Git Bash onto your system.
2. Download or git clone the Senior-Project-Spring-2026 package.
3. Open the package's folder as your Django project in the IDE of your choice.
4. Download Django packages: **pip install django**

You may start a virtual environment (venv) to run the project.
Using Visual Studio Code, you can host your own local version of the Django project.
Using a web server, you can host the project on an easily-accessible URL.

## Usage
The project is intended for testing patients for cognitive decline. 
Patients will access the test area to take a submit tests. 
Doctors can access these tests and view results for medical purposes.

## Project Goals
Our project’s core goals are to:
1. Provide a web-based application that allows PPST assessments to
be easily administered, accessed, and analyzed online.
2. Develop a stable, scalable system supported by a dynamic
database to ensure long-term reliability and maintainability.
